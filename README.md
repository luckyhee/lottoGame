# lottoGame
로또 게임  앱
 개발 언어 : 코틀린
-------------------------------------


1. 다섯 게임이 있으며 사용자는 스피너를 선택해 원하는 번호를 선택 후 결과 버튼을 누르면 당첨 번호가 출력
2. 다음 줄에는 사용자의 로또 결과가 뜬다
(예시 : 1번 꽝! 2번 꽝! 3번 꽝! 4번 꽝! 5번 꽝!)
3. 초기화 버튼을 클릭 시 게임은 다시 시작된다
